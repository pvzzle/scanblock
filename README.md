# Scanblock

## Проблема

Блокчейн Ethereum открыт всем, поэтому у каждого пользователя есть возможность получить необходимую ему информацию о конкретном блоке, транзакции и т.д. Scanblock предоставляет набор простых инструментов, сильно упрощающих жизнь интересующимся.

## Требования (покрытие кейсов, черновик)

- Whale alert: приложение постоянно анализирует успешно прошедшие транзакции и может уведомлять о крупных переводах по запросу пользователя. Позволяет фиксировать действия "крупных игроков" (для дальнейшей интерпретации) / узнавать о появлении новых "китов".
- Штрилиц: приложение может "следить" за действиями конкретных участников сети. Узнай, сколько долларов в месяц теряет твой друг-криптотрейдер.
- Формат "Я написал свой смарт-контракт, и мне шок как важно отслеживать проходящие по нему сделки в режиме реального времени": для энтузиастов и тех, кто знает, что делает. Приложение отслеживает и уведомляет обо всех операциях со смарт-контрактами по заданным адресам.
